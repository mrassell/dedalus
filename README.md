## Customization

Edit `src/index.css` to customize the color palette and typography:

```css
:root {
  --color-bg: #0a0908;
  --color-accent: #e8a849;
  --font-display: 'Instrument Serif', Georgia, serif;
  --font-body: 'Outfit', sans-serif;
}
```
